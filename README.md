# 五子棋網頁遊戲 (Gomoku Web Game)

一個使用純 HTML5、CSS3 和 JavaScript 開發的單頁面五子棋遊戲，支援雙人對弈。

## 🎮 遊戲特色

- ✨ 標準 15×15 棋盤
- 🎯 完整的五子棋規則實現
- 🖱️ 直觀的滑鼠點擊操作
- 🔄 悔棋功能
- 🏆 自動勝利檢測（橫、豎、斜四個方向）
- 💫 獲勝棋子高亮顯示
- 📱 響應式設計，支援多種螢幕尺寸

## 🚀 快速開始

### 線上遊玩

1. 下載 `index.html` 檔案
2. 使用瀏覽器直接開啟檔案
3. 開始遊戲！

### 本地部署

```bash
# 克隆專案
git clone https://github.com/Takming112/Team3.git

# 進入目錄
cd Team3

# 使用瀏覽器開啟 index.html
# 或使用本地伺服器（推薦）
python -m http.server 8000
# 然後在瀏覽器訪問 http://localhost:8000
```

## 📖 遊戲規則

### 基本規則

1. **先手**: 黑棋先行
2. **輪流下棋**: 黑白雙方輪流在棋盤交叉點落子
3. **勝利條件**: 率先在橫、豎、斜任一方向形成連續五顆棋子即獲勝
4. **棋子固定**: 棋子放下後不可移動

### 操作說明

- **落子**: 點擊棋盤上的交叉點
- **重新開始**: 點擊「重新開始」按鈕重置遊戲
- **悔棋**: 點擊「悔棋」按鈕撤銷上一步

### 核心資料結構

```javascript
// 遊戲配置
CONFIG = {
    BOARD_SIZE: 15,      // 棋盤大小
    CELL_SIZE: 40,       // 格子大小
    STONE_RADIUS: 12,    // 棋子半徑
    BLACK: 1,            // 黑棋
    WHITE: 2,            // 白棋
    EMPTY: 0             // 空位
}

// 遊戲狀態
gameState = {
    board: Array[15][15],    // 棋盤陣列
    currentPlayer: 1,         // 當前玩家
    isGameOver: false,        // 遊戲是否結束
    winner: null,             // 獲勝者
    moveHistory: [],          // 歷史記錄
    winningLine: []           // 獲勝線
}
```

### 核心函式

| 函式名稱 | 功能說明 |
|---------|---------|
| `initGame()` | 初始化遊戲狀態與棋盤 |
| `drawBoard()` | 繪製棋盤與棋子 |
| `drawStone(x, y, player)` | 繪製指定位置的棋子 |
| `placeStone(x, y)` | 處理落子邏輯 |
| `checkWin(x, y, player)` | 檢查是否獲勝 |
| `countDirection(x, y, dx, dy, player)` | 計算指定方向連續棋子數 |
| `undoMove()` | 悔棋功能 |
| `resetGame()` | 重新開始遊戲 |
| `updateUI()` | 更新介面顯示 |
